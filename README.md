# EPUB 3.3規格文件中文翻譯版本

## 文件說明

本清單中之文件，是台灣數位出版聯盟基於W3C EPUB 3 Working Group所推出的EPUB 3.3規格群，2022年10月4日編輯草稿之中文翻譯版。
EPUB 3.3版本為第一次由W3C工作小組循W3C推薦流程（W3C Recommendation Track）制定的標準，未來將會成為正式標準。
本文件也預期由台灣數位出版聯盟提交正式中文翻譯版。
